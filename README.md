# Example Projects for QueryDSL EntityQL

This repository is a part of [EntityQL](https://github.com/eXsio/querydsl-entityql) project.
It is a set of Example Projects that demonstrate all possible use cases of EntityQL. 

There are so many different ways you can use EntityQL, that describing them all here would be inefficient and impractical.
Below you can find usage matrix with links to example projects:
